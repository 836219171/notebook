# 求解next_permutation

### 题意

给一个排列A（可能含有相同数字），求其全排列（按字典序）中的下一个，即next_permutation

如果A是全排列中最后一个，则求全排列第一个

### 条件范围

N : 未知，可以认为是 10^5

Ai : 未知，可以认为是 int32

### 样例输入1
```
[1,2,3]
```

### 样例输出1
```
[1,3,2]
```

### 样例输入2
```
1 3 4 2
```

### 样例输出2 
```
1 4 2 3
```

### 样例输入3
```
1 3 3 1
```

### 样例输出3
```
3 1 1 3
```
