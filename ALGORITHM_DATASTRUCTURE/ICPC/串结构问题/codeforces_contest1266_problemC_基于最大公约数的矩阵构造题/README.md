# 基于最大公约数的矩阵构造题

### 题意

构造一个矩阵A，长为R，宽为C

设R行及C列的最大公约函数组合成一个数组B（长度为R+C）

满足几个条件
- 要求B是distinct的，即不能有重复数字
- 要求B的最大值最小
- 要求A的数值范围在[1, 10^9]

如果构造不出来，输出 0

### 条件范围

R : [1, 500]

C : [1, 500]

### 样例输入1

```
2 2
```

### 样例输出1

```
4 12
2 9
```

### 样例输入2

```
1 1
```

### 样例输出2

```
0
```
