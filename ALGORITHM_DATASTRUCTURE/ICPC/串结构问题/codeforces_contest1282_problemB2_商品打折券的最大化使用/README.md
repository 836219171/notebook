# 商品打折券的最大化使用

### 题意

T组输入

有N个商品，每个商品价格为Ai（块），手上有P块钱

有任意多个打折券，功能是选择 K 个商品，只付最贵的商品钱

因此，购买方式有两种（可以使用任意次数），如下：

- 不使用打折券，则一次买一个商品，买完后 P := P - Ai
- 使用打折券，则选择 K 个商品，只付其中最贵的，买完后 P := P - Amax（Amax是选定的K个商品中最高价）

要求使用最优策略，令购买到的商品数量最多

### 条件范围

T : [1, 10^4]

N : [2, 2 * 10^5]

SUM(N) for all T : [2, 2 * 10^5]

Ai : [1, 10^4]

P : [1, 2 * 10^9]

K : [2, N]
