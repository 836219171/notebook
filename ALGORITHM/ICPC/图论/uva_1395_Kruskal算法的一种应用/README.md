
# Kruskal算法的一种应用

### 题意

给一个简单图（无重编、无自环），N个点，M条边，每条边边权是Wi

求一个最小生成树，使得树中【Wi的最大值 - Wi的最小值】的结果最小，输出该结果

如果图构成不了一棵树，输出 -1

多CASE，以0 0结尾

### 条件范围

N ：[2, 100]

M : [0, N * (N-1) / 2]

Wi : [1, 10000]

### 样例输入1
```
4 5
1 2 3
1 3 5
1 4 6
2 4 6
3 4 7
```

### 样例输出1
```
1
```
