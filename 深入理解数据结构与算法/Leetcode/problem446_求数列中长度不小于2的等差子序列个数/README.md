
# 求数列中长度不小于2的等差子序列个数

# 题目
### 题意
```
给一个数列A，长度为N
求有多少个子序列满足
（1）长度不小于2
（2）是等差数列
```

### 条件范围
```
N: [0, 1000]
A[i]: [-2^31, 2^31-1]
保证答案小于 2^31 - 1
```

### 样例输入1
```
[2, 4, 6, 8, 10]
```

### 样例输出1
```
7
```

### 样例解释1
```
[2,4,6]
[4,6,8]
[6,8,10]
[2,4,6,8]
[4,6,8,10]
[2,4,6,8,10]
[2,6,10]
```

### 样例输入2
```
[3, 2, 1, 1, 2, 3]
```

### 样例输出2
```
4
```

### 样例解释2
```
[3, 2, 1]
[3, 2, 1]
[1, 2, 3]
[1, 2, 3]
```
