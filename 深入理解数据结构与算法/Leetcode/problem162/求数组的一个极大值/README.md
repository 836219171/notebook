# 求数组的一个极大值

### 题意
```
给一个数组A，长度为N，求一个位置 i ，要求比位置 i-1 的元素大 且 比位置 i+1 的元素大

数组相邻两个位置元素不相等，可以假设 -1 和 n 两个边界位置是负无穷

如果有多个答案，返回任意一个即可
```

### 条件范围
```
要求算法复杂度为O(log)
```

### 样例输入
```
[1,2,3,1]
[1,2,1,3,5,6,4]
```

### 样例输出
```
2
1或5
```
