# 源码阅读笔记

### 阅读方法
- 拿到源码链接及相关文档/视频（若无源码，不继续）
- 下载源码，并在IntelliJIDEA中尝试运行Main方法（若运行不成功，尝试解决）
- 根据源码内容及相关文档/视频，提出问题，尝试解决之，并做记录。重复这一步
- 分析源码
  - 查看pom.xml，关注dependency以及未见过的标签
  - 分析项目各代码模块、关键类及方法的功能
