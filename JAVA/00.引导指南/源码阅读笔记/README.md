# 源码阅读笔记

### 阅读方法
- 拿到文章及源码链接（若无源码，不继续）
- 下载源码，并在IntelliJIDEA中尝试运行Main方法（若运行不成功，尝试解决）
- 根据文章及源码内容，提出问题，尝试解决之，并做记录。重复这一步
- 阅读源码步骤
  - 查看pom.xml，关注dependency以及未见过的标签
  - 分析项目代码模块及功能作用
  - 分析关键类及方法功能作用
  - 做记录
