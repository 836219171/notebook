# 四种基础链表

### 单向链表

1 -> 2 -> 3

每项指向后继，最后一项无后继

### 双向链表

1 <-> 2 <-> 3

每项指向后继及前驱，第一项无前驱，最后一项无后继

### 单向循环链表

1 -> 2 -> 3 -> (1)

每项指向后继，最后一项有后继

### 双向循环链表

1 <-> 2 <-> 3 <-> (1)

每项指向后继，第一项有前驱，最后一项有后继
